# 实习周报网页

这是一个可部署到 GitHub 和 Netlify 的静态网页项目。

## 文件结构

- `index.html`：网页内容、周报模块、图片引用路径
- `style.css`：页面样式和移动端适配
- `assets/WEEK 1/`：第一周项目图片
- `assets/WEEK 2/`：第二周项目图片

## 更新图片时是否需要改 index.html

如果只是替换图片，并且文件夹、文件名、后缀完全不变，不需要修改 `index.html`。

如果新增图片、删除图片、修改文件名、修改后缀，或把图片移动到其他周文件夹，就需要同步修改 `index.html` 里的图片路径。
